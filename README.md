# Serve static website using Node.js and Express web server

## Dependencies

What you need to run this app:

- [Node.Js](https://nodejs.org)

## Instructions

### Setup

- 1 - Clone this repo:

```
git clone
```

- 2 - Copy/paste static website in /public

* 3 - Run this command:

```
npm install
```

- 4 Run this command:

```
npm start
```

- 4 Point your browser to:

```
http://localhost:3000
```
